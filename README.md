\# 🌾 Predicting Agriculture Using Naive Bayes Algorithm



A Machine Learning based web application that helps farmers by 

predicting suitable crops and estimating crop yield using AI.



\## 📌 Features

\- 🌱 Crop Prediction using Gaussian Naive Bayes algorithm

\- 📊 Yield Prediction using Random Forest Regressor

\- 🌤️ Weather Forecast based on location

\- 📁 CSV file upload and data preview



\## 🛠️ Tech Stack

\- Python 3.8

\- Flask (Web Framework)

\- Scikit-learn (Machine Learning)

\- Naive Bayes \& Random Forest algorithms

\- HTML, CSS, Bootstrap (Frontend)



\## 📈 Model Accuracy

\- Yield Prediction: 96%

\- Crop Prediction: 59%
